Distribuições EdxLinux e FostonLinux

EdxLinux

EdxLinux é uma distribuição Linux voltada para a educação, oferecendo um ambiente rico em recursos educacionais e ferramentas de aprendizado.

FostonLinux

FostonLinux é uma distribuição Linux focada em desempenho e estabilidade, ideal para usuários avançados e desenvolvedores.

Script de Instalação do Cubic

Para instalar o Cubic (Custom Ubuntu ISO Creator) em qualquer uma das distribuições acima, execute o seguinte script:

#!/bin/bash
# Atualizar a lista de pacotes
sudo apt update

# Instalar dependências necessárias
sudo apt install -y python3-pyqt5 python3-pyqt5.qtsvg genisoimage

# Baixar o Cubic
wget https://launchpad.net/cubic/+download

# Instalar o Cubic
sudo dpkg -i cubic*.deb

# Corrigir dependências
sudo apt install -f

# Verificar a instalação
cubic --version
