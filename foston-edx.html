Instalação de Programas no EdxLinux e FostonLinux

Bem-vindo ao guia de instalação de programas no EdxLinux e FostonLinux. Siga os passos abaixo para configurar seu sistema corretamente.

1. Clonar o Repositório

Primeiro, você precisará clonar o repositório do GitHub que contém os scripts de instalação:

git clone https://github.com/oliveiraezequiel/fostonlinux-e-edxlinux.git

2. Navegar para o Diretório do Repositório

Depois de clonar o repositório, navegue para o diretório clonado:

cd fostonlinux-e-edxlinux

3. Executar o Script de Instalação

Agora você pode executar o script de instalação. Este script instalará os programas necessários no seu sistema:

sudo bash install.sh

4. Verificar a Instalação

Para garantir que tudo foi instalado corretamente, você pode verificar os programas instalados:

sudo bash verify.sh
